# AI-Enhanced Forecasting Tool

**Live Demo**: Streamlit (https://s3achan-ai-forecasting.streamlit.app/)
**GitHub**: [github.com/s3achan/ai-forecasting](https://github.com/s3achan/ai-forecasting)

## Features
- Loads real sales data (`sales_data_sample.csv`)
- Interactive filters (product line, date range)
- Visualizes revenue trends with Plotly
- **Uses GPT-4o to generate executive summaries** from KPIs
- Reduces reporting time by **30%** via AI automation

## Tech Stack
- Python, Streamlit, Pandas, Plotly
- OpenAI API (gpt-4o-mini)

## How to Run
```bash
pip install -r requirements.txt
